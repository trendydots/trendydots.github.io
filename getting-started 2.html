Getting started with the trendy dots - trend research platform β

Making a new report

A report allows you to input up to 5 keywords. Keywords are semantically annotated and the platform performs queries to retrieve several publicly available data sets for each keyword.

View. Deduce. Take conclusions

There are several applications integrated in the platform which allow you see data related to your keywords. On the right hand side, there are fine-tuning fields which allow you to experiment with headlines, note conclusions, or save a default time interval for which will be read by the applications next time you explore the report.
